Mobile Game Project Naturia

Naturia

Naturiaは、自然や生き物をテーマにしたiOS・Android向けモバイルゲームプロジェクトです。 生物収集や探索をテーマにしたゲーム体験の提供を目指し、現在開発を進めています。
